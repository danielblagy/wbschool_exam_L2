Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
Программа выведет:
[77 78 79]

Берем сабслайс начиная с элемента 1 (2-й по счету элемент в слайсе a) и до 4-го элемента,
не включая его.

```
